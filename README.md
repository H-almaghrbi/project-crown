# 👑 Project Crown - منصة تعليمية

منصة تعليمية متقدمة مبنية على **WordPress** لإدارة الدورات التعليمية والمحتوى التعليمي بكفاءة عالية.

## 🎯 نبذة عن المشروع

**Project Crown** هو منصة تعليمية شاملة توفر:
- 📚 إدارة الدورات والمحاضرات
- 👥 نظام إدارة الطلاب والمدرسين
- 📊 تتبع التقدم الأكاديمي
- 💬 تفاعل بين الطلاب والمدرسين
- 📝 إدارة المحتوى التعليمي

---

## ✨ المميزات الرئيسية

- ✅ واجهة سهلة الاستخدام
- ✅ نظام تقييم وتقويم متقدم
- ✅ إدارة فئات الدورات والمحاضرات
- ✅ نظام المستخدمين والصلاحيات
- ✅ دعم اللغة العربية بالكامل
- ✅ قابلة للتوسع والتطوير

## 🛠️ المتطلبات

### Server Requirements:
- **PHP:** 7.4 أو أحدث
- **MySQL:** 5.7 أو أحدث
- **WordPress:** 5.0 أو أحدث
- **Apache/Nginx** مع دعم `.htaccess` (للـ Apache)

### المكتبات المطلوبة:
- WordPress Core
- PHP Extensions: `mysqli`, `curl`, `json`

## 📥 خطوات التثبيت

### 1. استنساخ المستودع
```bash
git clone https://github.com/H-almaghrbi/project-crown.git
cd project-crown
```

### 2. تثبيت WordPress
```bash
# تأكد من وجود خادم محلي (مثل XAMPP أو WAMP)
# ضع المشروع في مجلد htdocs (أو المجلد المناسب لخادمك)
```

### 3. إنشاء قاعدة البيانات
```bash
# أنشئ قاعدة بيانات جديدة
mysql -u root -p -e "CREATE DATABASE project_crown;"
```

### 4. تكوين الملف wp-config.php
```php
define('DB_NAME', 'project_crown');
define('DB_USER', 'root');
define('DB_PASSWORD', 'your_password');
define('DB_HOST', 'localhost');
```

### 5. الوصول إلى التطبيق
```
http://localhost/project-crown
```
